# SourceWebpackModule

## **将Cesium源码及依赖的第三方包使用Webpack打包压缩并混淆以便于发布为可用的SDK**

## 该项目基于Cesium-1.96版本进行改造

## 运行要求

### 1. nodejs的版本为10至12

### 2. gulp的版本为4.0.2

---

## 使用方法

### 执行如下代码安装依赖

```javascript
    // 安装依赖
    npm install --registry=https://registry.npm.taobao.org

    // 执行package.json的scripts命令对项目进行按需打包
```
